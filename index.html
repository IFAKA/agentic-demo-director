Realistic flow

The whole workflow follows the same loop as the hero.

1

Type the request

A normal prompt starts it: “make the launch demo video” or “record the onboarding walkthrough.”

2

Watch the recipe stream

The agent detects routes, fixtures, selectors, hooks, and writes the repeatable `.demo.ts` recipe.

3

Ask for a tweak

“Make the delete slower” changes the exact gesture timing instead of editing pixels by hand.

4

Get the video

Playwright performs the demo, ffmpeg composes the MP4, and ffprobe validates the result.

Install

One local CLI turns an app repo into a demo workspace.

`init` adds the demo folder and scripts. After that, natural-language requests become scenario edits and recorder runs. 

npm install --save-dev ../agentic-demo-director/packages/cli
npx demo-director init

npm run demo:doctor
npm run demo:record
npm run demo:inspect

Generated recipe

The agent keeps the demo as code, so every tweak is repeatable. 

import { defineDemo } from "@faka/demo-director";

export default defineDemo({
  name: "main",
  startUrl: "/demo/launch?scene=intro",
  output: "dist/launch-video/idlediary-launch-4x5.mp4",
  format: {
    width: 1080,
    height: 1350,
    fps: 30,
    duration: { min: 28, max: 40 },
    crop: "social-4x5",
    cursor: "hidden",
  },
  steps: async ({ page, gesture, expect }) => {
    await gesture.tap(page.getByRole("button", { name: "Start recording" }));
    await gesture.drag({
      from: page.locator("[data-clip-id] button").nth(1),
      to: page.getByTestId("review-action-bar"),
      durationMs: 950,
    });
    await expect(page.getByRole("heading", { name: "4 Tiny Moments" })).toBeVisible();
  },
});
